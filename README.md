# zu-viele-mangas

Website für den deutschen Anime & Manga Podcast **zu-viele-mangas** von Oskar und Elias.

Statische HTML/CSS-Seite — keine Build-Pipeline. Einfach `index.html` im Browser öffnen oder per GitHub Pages hosten.

## Struktur

- `index.html` — Home (Hero, Podcast-Beschreibung, neueste Folgen, Host-Bios)
- `ueber-uns.html` — Über uns + alle Kanäle
- `merchandise.html` — SpreadShop-Einbindung
- `impressum.html`, `datenschutz.html` — Rechtsseiten (noch mit TODO-Platzhaltern)
- `assets/` — CSS, JS, Logo, Favicon
