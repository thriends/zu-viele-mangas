2. Hosting und Server-Logfiles

Diese Website wird gehostet bei GitHub Pages, einem Dienst der GitHub, Inc. (88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA), einer Tochtergesellschaft der Microsoft Corporation.

Beim Aufruf dieser Website erhebt GitHub in sogenannten Server-Logfiles automatisch Informationen, die dein Browser übermittelt. Dies sind: Browsertyp und Browserversion, verwendetes Betriebssystem, Referrer-URL, Hostname des zugreifenden Rechners, Uhrzeit der Serveranfrage und IP-Adresse. Diese Daten werden nicht mit anderen Datenquellen zusammengeführt. Die Erfassung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an fehlerfreier Darstellung und Sicherheit).

Für die Datenübermittlung in die USA besteht eine Grundlage über den EU-US Data Privacy Framework. Weitere Informationen: GitHub Privacy Statement.

3. Cookies

Wir setzen selbst keine Cookies. Diese Website verwendet weder eigene Cookies noch Tracking-, Analyse- oder Marketing-Tools. Es gibt keinen Cookie-Banner, weil es nichts zum Zustimmen gibt — bis du aktiv mit dem Shop interagierst (siehe Abschnitt 6).

4. Schriftarten

Schriftarten (Bricolage Grotesque, Inter) werden direkt von unserem Server (GitHub Pages) ausgeliefert. Es findet keine Übertragung an Google Fonts, gstatic.com oder ähnliche Drittanbieter statt.

5. Podcast-Folgen

Die Liste der neuesten Folgen auf der Startseite wird aus einer statischen JSON-Datei auf unserem Server geladen. Diese Datei wird serverseitig alle sechs Stunden über die öffentliche iTunes-Lookup-API von Apple aktualisiert — dabei findet die Kommunikation zwischen unserem Build-Server (GitHub Actions) und Apple statt, nicht zwischen deinem Browser und Apple. Apple erfährt also nicht, dass du unsere Seite besucht hast.
